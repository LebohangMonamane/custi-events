Everything You Need to Run a Seamless Event

A complete WhatsApp-powered toolkit — from event creation to guest communication and beyond.

Platform Overview

Every feature is built to help organisers set up faster, communicate better, and manage guests with less friction.

Custi Events combines organiser-side controls with a WhatsApp-first guest experience, so you can handle setup, operations, and communication from one place without forcing guests into another app.

Organiser-first controls Keep setup, guest data, and operations in one portal
WhatsApp-native communication Let guests respond inside the channel they already use
Built for active event management Handle announcements, seating, and messaging as the event moves
Core Features

Capabilities That Cover the Full Event Workflow

From first setup to guest replies and team coordination, these are the tools organisers use to run an event smoothly on Custi Events.

Setup Guided Flow

Event Setup Wizard

A guided step-by-step wizard walks you through creating your event — name, date, venue, capacity, categories, and all configuration options.

Launch with a clean event structure from day one
Communication Event Channel

WhatsApp Number Assignment

Assign a dedicated WhatsApp Cloud API number to your event. All guest interactions flow through this number, keeping everything organised.

Keep every RSVP and reply connected to one event line
Guest Response Automated RSVP

RSVP Automation

Guests receive an invitation via WhatsApp and RSVP with a simple reply. No forms, no apps, no friction — just a message.

Reduce response friction for every guest
Operations Seating Control

Seating Management

Assign guests to tables and seats from the organiser portal. Import seating plans via CSV or manage them manually.

Move from guest list to seating plan without extra tools
Messaging Announcements

Messaging & Announcements

Send targeted messages, event reminders, and announcements to all guests or specific segments — directly via WhatsApp.

Keep guests informed before and during the event
Guest Data Bulk Import

Bulk Guest Import

Upload your entire guest list via CSV in seconds. Validate, review, and confirm before sending invitations.

Bring existing guest lists into the platform quickly
Control Center Dashboard

Organiser Portal

A full-featured web dashboard to manage events, guests, messages, seating, and settings — all in one place.

Manage the full event operation from one workspace
Outreach Broadcasts

Broadcast Messages

Send bulk WhatsApp messages to your entire guest list or filtered groups with a single action.

Reach everyone or target specific guest segments fast
Team Access Roles

Role-Based Access

Invite team members with specific roles — administrators, support agents, and more — to collaborate on your event.

Let teams collaborate without losing control

The result: organisers get one connected toolkit for setup, messaging, guest operations, and team coordination, all centered around WhatsApp.

Powered by WhatsApp Cloud API

Custi Events integrates directly with the WhatsApp Cloud API, giving your guests a native WhatsApp experience. No third-party apps, no new accounts — just the app they already use every day.
