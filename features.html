Core Features

Built for Organisers, Loved by Guests

Every feature is designed to reduce friction for organisers and make the guest experience effortless.

Event Setup Wizard

A guided step-by-step wizard walks you through creating your event — name, date, venue, capacity, categories, and all configuration options.

WhatsApp Number Assignment

Assign a dedicated WhatsApp Cloud API number to your event. All guest interactions flow through this number, keeping everything organised.

RSVP Automation

Guests receive an invitation via WhatsApp and RSVP with a simple reply. No forms, no apps, no friction — just a message.

Seating Management

Assign guests to tables and seats from the organiser portal. Import seating plans via CSV or manage them manually.

Messaging & Announcements

Send targeted messages, event reminders, and announcements to all guests or specific segments — directly via WhatsApp.

Bulk Guest Import

Upload your entire guest list via CSV in seconds. Validate, review, and confirm before sending invitations.

Organiser Portal

A full-featured web dashboard to manage events, guests, messages, seating, and settings — all in one place.

Broadcast Messages

Send bulk WhatsApp messages to your entire guest list or filtered groups with a single action.

Role-Based Access

Invite team members with specific roles — administrators, support agents, and more — to collaborate on your event.

Powered by WhatsApp Cloud API

Custi Events integrates directly with the WhatsApp Cloud API, giving your guests a native WhatsApp experience. No third-party apps, no new accounts — just the app they already use every day.
