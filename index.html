What Custi Events Is

An organiser platform for event setup and WhatsApp automation

  • Event setup platform for weddings, conferences, church events, fundraisers, and corporate functions
  • WhatsApp-first RSVP and guest communication workflow
  • SaaS product for organisers managing guests, seating, announcements, and event operations
What Custi Events Is Not

Not a public guest-facing event website

  • Not a ticketing platform for selling event access online
  • Not a guest website where attendees browse event pages and RSVP publicly
  • Not a public RSVP microsite replacing WhatsApp-based guest interaction
