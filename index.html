Features

Everything You Need to Run a Seamless Event

A complete toolkit for event setup, guest management, and WhatsApp-powered communication.

Event Setup Wizard

Guided step-by-step wizard to create your event — name, date, venue, capacity, and all configuration options.

RSVP Automation

Guests confirm attendance directly in WhatsApp with a simple reply — no forms, no apps, no friction.

Seating Management

Assign guests to tables and seats from the organiser portal. Import via CSV or manage manually.

Messaging, Announcements & Broadcasts

Send targeted messages, reminders, and bulk broadcasts to all guests or specific segments directly via WhatsApp.

Bulk Guest Import

Upload your entire guest list via CSV in seconds. Validate and confirm before sending invitations.

Organiser Portal

A full-featured dashboard to manage events, guests, messages, seating, and settings — all in one place.
